Historia do HTML

(voltar)

O HTML foi criado em 1991, por Tim Berners-Lee, no CERN (European Council for Nuclear Research) na suíça. Inicialmente o HTML foi projetado para interligar instituições de pesquisa próximas, e compartilhar documentos com facilidade. Em 1992, foi liberada a biblioteca de desenvolvimento , WWW ( World Wide Web), uma rede de alcance mundial, que junto com o HTML proporcionou o uso em escala mundial da WEB.

As primeiras versões do HTML foram definidas com regras sintáticas flexíveis, o que ajudou aqueles sem familiaridade com a publicação na Web. Através do tempo, a utilização de ferramentas para autoria de HTML aumentou, assim como a tendência em tornar a sintaxe cada vez mais rígida. Apesar disso, por questões históricas (retrocompatibilidade), os navegadores ainda hoje conseguem interpretar páginas web que estão longe de ser um código HTML válido.

Como funciona o HTML

(voltar)

HTML (Linguagem de marcação de Hiper-texto) , é responsável por dizer ao seu navegador como o conteúdo vai estar estruturado. Ele é a linguagem que pega o conteúdo e organiza na tela, sem o HTML a web seria um estilo muito rustica ainda que por exemplo nem link existiria.

Além da formatação de textos o HTML ainda tem papel fundamental na inclusão de imagens no texto, definindo onde e quando serão apresentadas, outra função do HTML é a inclusão de tabelas, ele vai além daquilo que se vê se umas páginas contem metadados para outras ferramentas é o HTML que cuida disso.

Tags e sua importancia

(voltar)

O HTML usado para formar qualquer objeto identificável, como uma lista, parágrafo, campos de formulários, vídeos, imagens e etc.

O navegador entende essas marcações que são invisíveis para o usuário normal e transforma o documento na página que podemos ver em qualquer site

  • Tags Relevantes

  1. Tag < a >

    A < a > tag define um hiperlink, que é usado para vincular de uma página a outra.

    O atributo mais importante do < a > elemento é o "href" atributo, que indica o destino do link.

  2. Tag < p >

    A < p >tag define um parágrafo.

  3. As tags < h1 >a < h6 >

    são usadas para definir cabeçalhos HTML

  4. Tag < img >

    A < img > tag é usada para incorporar uma imagem em uma página HTML .

  5. Tag < mark >

    A < mark >tag define o texto que deve ser marcado ou destacado.

  6. Tag < blockquote >

    7. A < blockquote >tag especifica uma seção citada de outra fonte.
